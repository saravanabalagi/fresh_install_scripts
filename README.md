## Fresh Install Scripts

This repo contains scripts that are useful to install a bunch of dev stuff on a freshly installed Ubuntu.

## Install

Run `sh all.sh`

It will install the following tools:

- remote tools
    - openssh
    - rdp

- dev utils and activity monitors 
    - dev_utils
        - curl
        - wget
        - git
        - vim
        - tmux
    - activity_monitors
        - htop
        - iotop
        - nvtop

- container tools
    - docker
    - nvidia-docker2
    - singularity
        - installs Go locally

- shell
    - zsh
    - oh-my-zsh

- ui
    - gnome_tweaks
